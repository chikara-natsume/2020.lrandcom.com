# Project overview

- Purpose: Next.js-based marketing/blog site. The README references a Notion-backed blog example; the current user notes the site uses microCMS for content.
- Tech stack: Next.js 12, React 18, TypeScript, styled-components, Redux Toolkit, axios, gsap, prismjs, react-svg; Node 22; pnpm.
- Repo structure (top-level):
  - `pages/` Next.js routes
  - `components/` UI components
  - `layouts/` layout wrappers
  - `lib/`, `utils/`, `hooks/` shared logic
  - `api/` (likely API utilities)
  - `assets/`, `public/` static assets
  - `store/` Redux store
  - `types/` TS types
