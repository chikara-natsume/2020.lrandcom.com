# Style and conventions

- Formatting: Prettier via `@soichiro_nitta/prettier-config` (see `.prettierrc.js`).
- Linting: ESLint via `@soichiro_nitta/eslint-config/presets/react-typescript-prettier` (see `.eslintrc.js`).
- Styling lint: Stylelint via `@soichiro_nitta/stylelint-config` (see `.stylelintrc.js`).
- Language: TypeScript + React (TSX in `pages/`, `components/`).
